개발자 자기소개 웹페이지를 처음부터 만들어줘.



\## 프로젝트 설정

\- 프로젝트명: roll-my-intro-site

\- 기술스택: Vue 3 + Quasar Framework (Vite 기반) + TypeScript

\- 린팅/포맷: ESLint + Prettier

\- 패키지매니저: npm



\## 페이지 구성 (싱글 페이지)

\- 상세 구성은 적당히 알아서 만들어줘

\- 반응형 웹 (모바일/태블릿/데스크톱)



\## 배포

\- GitHub 레포 생성: {cheadev5831}/roll-my-intro-site

\- GitHub Pages로 배포 (gh-pages 브랜치 방식)



\## 브랜치 전략

\- main: 배포용

\- develop: 통합

\- feature/plan, feature/design, feature/dev, feature/test



\## 커밋 규칙

작업 단계별로 커밋을 나눠서 해줘

각 커밋 메시지는 한글로, 작업 내용을 명확히 적어줘.
